# Template Guide

Welcome to the template guide! This section will walk you through how to use this template to create high-quality course material.

## Getting Started

To work with this template locally, follow these steps:

### 1. Prerequisites
Ensure you have **Node.js** (v18 or higher) installed on your machine.

### 2. Installation
Clone the repository and install the dependencies:

```bash
# Clone the repository
git clone https://github.com/BeyensJ/vuepress-course-template.git

# Enter the directory
cd vuepress-course-template

# Install dependencies
npm install
```

### 3. Development
Run the local development server to see your changes in real-time:

```bash
npm run docs:dev
```
The site will be available at `http://localhost:8080` (or the port specified in your terminal).

### 4. Production Build
To generate the static files for deployment:

```bash
npm run docs:build
```
The output will be in the `docs/.vuepress/dist` directory.
